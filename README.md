# OngekiFunctions

#### [オンゲキネット](https://ongeki-net.com/ongeki-mobile/)を用いて使える便利機能開発Gitです。

以下の機能の実装を予定しています。

- スコアデータ収集

  各枠（べ枠 リ枠 新曲枠 候補枠）の一覧の閲覧

  フィルター別ソート

- オンゲキネット上での画像化

  スクショに収まらないサイズの画像化
